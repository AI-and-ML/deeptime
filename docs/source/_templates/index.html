Deeptime

Deeptime is a Python library for analysis of time series data. In particular, methods for dimensionality reduction, clustering, and Markov model estimation are implemented.

The API is similar to that of scikit-learn and offers basic compatibility to its tools via ducktyping.

Deeptime can be installed via pip (pip install deeptime) and is also available via conda on conda-forge (conda install -c conda-forge deeptime).

Clustering - reduce dimension and discretize datasets.
Dimensionality reduction - project timeseries onto dominant processes.
Markov state models - learn about the kinetics of discrete jump processes.
Hidden markov models.
SINDy - Find governing equations from data.
Data - Some toy datasets to illustrate algorithms and benchmark.
